Architecture

Under the Hood

A thoughtfully engineered extension ‚Äî built lean, built smart, built to last.

TypeScript + esbuild

Every line of GitIndex is authored in TypeScript ‚Äî giving full type safety and maintainability ‚Äî then compiled by esbuild into a single, lightning-fast content script with zero bloat.

SPA-Aware Navigation

GitHub is a single-page application ‚Äî it never fully reloads between pages. GitIndex intercepts every client-side navigation and seamlessly activates the right features for each view without missing a beat.

Modular by Design

Built on a self-written modular framework, each feature lives in its own isolated module and submodule. Adding or removing capabilities is surgical ‚Äî no spaghetti, no side effects, and lightning-fast iteration.

Why a single content script? GitHub's SPA architecture means traditional multi-file approaches break down fast ‚Äî pages change without a reload, and injecting scripts on every navigation is wasteful and fragile. By compiling everything into one optimised bundle, GitIndex loads once, listens for navigation events, and intelligently mounts only the modules relevant to the current page. The result: instant activation, zero redundancy, and a codebase that's genuinely easy to extend.
