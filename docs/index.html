Features

Everything you wish GitHub had

Small tweaks with a big impact on your daily workflow.

File Hover Actions

Instantly copy a file's relative path, full URL, or raw URL — or open it in a new tab — right from the repository file list without any extra clicks.

Copy path Copy URL Copy raw URL Open in tab

Edit with IDE

Open any repository in your preferred editor with a single click. Supports GitHub.dev, VS Code, and GitPod — choose the tool that fits your flow.

GitHub.dev VS Code GitPod

Repository Preview Card

Hover over any repository in search results to instantly see stars, forks, open issues, size, license, top language, and more — without ever leaving the page.

Stars & Forks Open Issues Language License

Language Donut Chart

GitHub's language bar gets a visual upgrade — a beautiful donut chart replaces the thin progress bar in the repository sidebar, making language breakdowns much easier to read at a glance.

Visual upgrade Sidebar Language breakdown
